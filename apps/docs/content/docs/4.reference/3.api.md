# API Reference

Code Please packages export TypeScript APIs for programmatic use.

## @pleaseai/code-format

Formatter orchestration API.

### initFromProject

Initialize formatters from project configuration.

```typescript
import { initFromProject } from '@pleaseai/code-format'

const formatter = await initFromProject('/path/to/project')
```

### formatFile

Format a single file.

```typescript
import { formatFile } from '@pleaseai/code-format'

const result = await formatFile('/path/to/file.ts', {
  projectPath: '/path/to/project'
})

if (result.formatted) {
  console.log(`Formatted with ${result.formatter}`)
}
```

### status

Get formatter status.

```typescript
import { status } from '@pleaseai/code-format'

const info = await status('/path/to/project')
console.log(info.availableFormatters)
```

## @pleaseai/code-lsp

LSP client API.

### LSPManager

Main class for LSP operations.

```typescript
import { LSPManager } from '@pleaseai/code-lsp'

const manager = new LSPManager('/path/to/project')

// Get diagnostics
const diagnostics = await manager.diagnostics('/path/to/file.ts')

// Get hover info
const hover = await manager.hover('/path/to/file.ts', 10, 5)

// Find definition
const definition = await manager.definition('/path/to/file.ts', 10, 5)

// Find references
const references = await manager.references('/path/to/file.ts', 10, 5)

// Get completions
const completions = await manager.completion('/path/to/file.ts', 10, 5)

// Rename symbol
const edits = await manager.rename('/path/to/file.ts', 10, 5, 'newName')

// Cleanup
await manager.shutdown()
```

### LSP Server Registry

Access available servers.

```typescript
import { getServerById, getServersForExtension, LSP_SERVERS } from '@pleaseai/code-lsp'

// Get all servers
console.log(LSP_SERVERS)

// Get server by ID
const tsServer = getServerById('typescript')

// Get servers for file extension
const servers = getServersForExtension('.ts')
```

## @pleaseai/dora

MCP server API (typically run via CLI).

### MCP Tools

Available MCP tools when running `dora serve`:

| Tool | Description |
|------|-------------|
| `lsp_diagnostics` | Get file diagnostics |
| `lsp_hover` | Get hover information |
| `lsp_definition` | Find definition |
| `lsp_references` | Find references |
| `lsp_completion` | Get completions |
| `lsp_workspace_symbol` | Search workspace symbols |

## Types

### DiagnosticResult

```typescript
interface DiagnosticResult {
  uri: string
  diagnostics: Diagnostic[]
}

interface Diagnostic {
  range: Range
  message: string
  severity?: DiagnosticSeverity
  code?: string | number
  source?: string
}
```

### HoverResult

```typescript
interface HoverResult {
  contents: MarkupContent | string
  range?: Range
}
```

### Location

```typescript
interface Location {
  uri: string
  range: Range
}

interface Range {
  start: Position
  end: Position
}

interface Position {
  line: number
  character: number
}
```

## Configuration Types

### Config

```typescript
interface Config {
  language?: 'en' | 'ko'
  ignore_patterns?: string[]
  formatter?: false | Record<string, FormatterItem>
  lsp?: false | Record<string, LspItem>
}

interface FormatterItem {
  disabled?: boolean
  command?: string[]
  extensions?: string[]
  environment?: Record<string, string>
}

interface LspItem {
  enabled?: boolean
  root?: string
  command?: string[]
}
```
