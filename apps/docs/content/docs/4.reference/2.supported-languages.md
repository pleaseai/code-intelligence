# Supported Languages

Code Please supports formatting and LSP diagnostics for many languages.

## Language Support Matrix

| Language | Formatter | LSP Server |
|----------|-----------|------------|
| TypeScript | Biome, Prettier | typescript-language-server |
| JavaScript | Biome, Prettier | typescript-language-server |
| Python | ruff, uv | pyright |
| Go | gofmt | gopls |
| Rust | - | rust-analyzer |
| Kotlin | ktlint | JetBrains Kotlin LSP |
| Dart | dart format | dart language-server |
| Vue | Prettier | @vue/language-server |
| Prisma | prisma format | @prisma/language-server |
| C/C++ | clang-format | - |
| Elixir | mix format | - |
| Zig | zig fmt | - |
| OCaml | ocamlformat | - |
| Terraform | terraform fmt | - |
| Gleam | gleam format | - |

## Auto-Download Servers

These servers are automatically downloaded when needed:

| Server | Source | Install Location |
|--------|--------|------------------|
| Kotlin LSP | JetBrains releases | `~/.cache/dora/kotlin-lsp/` |
| Dart LSP | Dart SDK archives | `~/.cache/dora/dart-sdk/` |
| Vue LSP | npm registry | Project `node_modules/` |
| Prisma LSP | npm registry | Project `node_modules/` |
| ESLint LSP | GitHub (vscode-eslint) | `~/.cache/dora/eslint-lsp/` |
