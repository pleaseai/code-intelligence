# Formatter Configuration

Configure which formatters to use and how they behave.

## Formatter Item Schema

Each formatter can have these options:

```yaml
formatter:
  <formatter-name>:
    disabled: false # Disable this formatter
    command: [cmd, args...] # Custom command ($FILE placeholder)
    extensions: [.ts, .js] # File extensions
    environment: # Environment variables
      KEY: value
```

## Built-in Formatters

### JavaScript/TypeScript

```yaml
formatter:
  biome:
    extensions: [.ts, .tsx, .js, .jsx, .json]

  prettier:
    extensions: [.ts, .tsx, .js, .jsx, .json, .md, .css]
```

### Python

```yaml
formatter:
  ruff:
    extensions: [.py]

  # Alternative: uv format
  uv:
    extensions: [.py]
```

### Go

```yaml
formatter:
  gofmt:
    extensions: [.go]
```

### Rust

```yaml
formatter:
  rustfmt:
    extensions: [.rs]
```

### Other Languages

```yaml
formatter:
  # Kotlin
  ktlint:
    extensions: [.kt, .kts]

  # Dart
  dart:
    extensions: [.dart]

  # Elixir
  mix:
    extensions: [.ex, .exs]

  # Zig
  zig:
    extensions: [.zig]

  # Prisma
  prisma:
    extensions: [.prisma]
```

## Custom Formatters

Add your own formatters:

```yaml
formatter:
  my-formatter:
    command: [my-tool, format, --write, $FILE]
    extensions: [.custom, .xyz]
    environment:
      MY_CONFIG: /path/to/config
```

The `$FILE` placeholder is replaced with the actual file path.

## Override Built-in Formatters

Override command or extensions for built-in formatters:

```yaml
formatter:
  biome:
    command: [bunx, '@biomejs/biome', format, --write, $FILE]
    extensions: [.ts, .tsx, .js, .jsx, .json, .jsonc]
```

## Disable Formatters

### Disable all

```yaml
formatter: false
```

### Disable specific

```yaml
formatter:
  prettier:
    disabled: true
```

## Examples

### Biome only (disable Prettier)

```yaml
formatter:
  prettier:
    disabled: true
  biome:
    extensions: [.ts, .tsx, .js, .jsx, .json]
```

### Use dprint instead of default formatters

```yaml
formatter:
  biome:
    disabled: true
  prettier:
    disabled: true
  dprint:
    command: [dprint, fmt, $FILE]
    extensions: [.ts, .tsx, .js, .jsx, .json, .md]
```
