# MCP Server (Dora)

**Dora** is an MCP (Model Context Protocol) server that provides LSP tools for AI-powered IDEs. It enables advanced code intelligence features beyond basic diagnostics.

## What is MCP?

[Model Context Protocol](https://modelcontextprotocol.io) is a standard for connecting AI models to external tools and data sources. Dora implements MCP to expose LSP operations as tools.

## Installation

```bash
npm install -g @pleaseai/dora
```

## Starting the Server

```bash
dora serve
```

The server runs on stdio, designed for integration with MCP clients.

## Available Tools

### Diagnostics

```
lsp_diagnostics(file_path)
```

Get all errors and warnings for a file.

### Hover

```
lsp_hover(file_path, line, character)
```

Get type information and documentation at a position.

### Definition

```
lsp_definition(file_path, line, character)
```

Find where a symbol is defined.

### References

```
lsp_references(file_path, line, character)
```

Find all usages of a symbol.

### Completion

```
lsp_completion(file_path, line, character)
```

Get code completion suggestions.

### Workspace Symbols

```
lsp_workspace_symbol(query)
```

Search for symbols across the workspace.

## Configuration

Environment variables:

| Variable | Description | Default |
|----------|-------------|---------|
| `DORA_PROJECT_PATH` | Project root directory | cwd |
| `DORA_TIMEOUT` | Request timeout in ms | 30000 |

## MCP Client Configuration

Add to your MCP client configuration:

```json
{
  "mcpServers": {
    "dora": {
      "command": "dora",
      "args": ["serve"]
    }
  }
}
```

## JetBrains Integration

Dora can also connect to JetBrains IDEs via the Serena plugin for symbol navigation and code intelligence.

## Use Cases

- **AI-powered code review** - Analyze code for issues
- **Smart refactoring** - Find and update all references
- **Code exploration** - Navigate definitions and usages
- **Completion assistance** - Get context-aware suggestions
