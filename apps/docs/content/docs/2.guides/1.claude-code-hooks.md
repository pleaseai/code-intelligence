# Claude Code Hooks

Code Please integrates with Claude Code through the [PostToolUse hook system](https://docs.anthropic.com/en/docs/build-with-claude/claude-code/hooks). This guide explains how hooks work and how to configure them.

## How Hooks Work

When Claude uses the `Write` or `Edit` tool to modify a file:

1. Claude Code triggers the `PostToolUse` hook
2. File information is sent to Code Please via stdin (JSON format)
3. Code Please processes the file (format and/or LSP check)
4. Results are returned as hook output
5. Claude sees the feedback and can respond

## Basic Configuration

Add to `.claude/settings.json`:

```json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Write|Edit",
        "hooks": [
          {
            "type": "command",
            "command": "npx @pleaseai/code format --stdin"
          },
          {
            "type": "command",
            "command": "npx @pleaseai/code lsp --stdin"
          }
        ]
      }
    ]
  }
}
```

## Hook Input Format

Code Please receives JSON input via stdin:

```json
{
  "tool_name": "Write",
  "tool_input": {
    "file_path": "/path/to/file.ts",
    "content": "..."
  }
}
```

## Hook Output Format

### Format Hook Output

The format hook is designed to be silent on success. When a file is formatted, no output is shown to minimize noise. The file is simply updated in place.

### LSP Hook Output

When errors are found:

```
✗ 3 errors found
  ✗ index.ts:15:7 [TS2322]: Type 'string' is not assignable to type 'number'
  ✗ index.ts:20:5 [TS2304]: Cannot find name 'undefined_var'
  ✗ utils.py:8:12: Undefined variable 'user'
```

When no errors:

```
(no output)
```

## Format Only

To use only auto-formatting (no LSP):

```json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Write|Edit",
        "hooks": [
          {
            "type": "command",
            "command": "npx @pleaseai/code format --stdin"
          }
        ]
      }
    ]
  }
}
```

## LSP Only

To use only LSP diagnostics (no formatting):

```json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Write|Edit",
        "hooks": [
          {
            "type": "command",
            "command": "npx @pleaseai/code lsp --stdin"
          }
        ]
      }
    ]
  }
}
```

## Environment Variables

| Variable | Description | Default |
|----------|-------------|---------|
| `CODE_PROJECT_PATH` | Project root directory | cwd |
| `CLAUDE_PROJECT_DIR` | Set by Claude Code in hook mode | - |

## Troubleshooting

### Hook not triggering

- Ensure the matcher regex matches `Write` or `Edit`
- Check that `@pleaseai/code` is installed globally or available via npx

### Errors not showing

- Verify the language server is installed and working
- Check that the file extension is supported
- Run `code lsp <file>` manually to test
